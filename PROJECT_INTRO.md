# 🧪 ByteAlchemy: 现代化极客加解密实验台

**ByteAlchemy** 是一款为安全研究员、CTF 选手及开发者打造的现代化加解密辅助工具。它将 **CyberChef** 的灵活性与原生 Python 算法的强大处理能力相结合，旨在解决“非标加密算法调试”这一痛点。

---

## 🌟 核心亮点

### 1. 🧩 拼图式操作链 (Operation Chain)
- **自由组合**：通过拖拽算子构建处理流程，支持编码（Base64, Hex）、加密（AES, SM4）及代码格式化。
- **实时调试**：修改任意步骤的参数，结果立等可见，支持一键开关特定步骤。

### 2. 🪄 魔改 S 盒 (Magic S-Box)
- **可视化编辑**：内置 16x16 矩阵视图，直观查看并修改 AES/SM4 的置换盒。
- **应对非标**：支持从标准算法克隆 S 盒并手动置换，配合底层原生 Python 驱动，轻松应对各类非标魔改算法。

### 3. 🎨 极客美学界面
- **现代 UI**：基于 Element Plus 与深紫色极客主题，视觉舒适且专业。
- **终端壁纸**：支持自定义终端背景图与透明度，打造独一无二的交互环境。

### 4. ⌨️ 强大的脚本库
- **交互式执行**：上传 Python 脚本并直接在内置终端运行。
- **智能对话框**：自动解析脚本 `input()` 调用，生成参数预填界面，彻底告别命令行输入延迟。

---

## 🛠️ 技术底座

- **前端**：Vue 3 + Vite + Element Plus + xterm.js
- **外壳**：Electron (提供跨平台桌面体验)
- **后端**：Python 3 + FastAPI + WebSockets (处理计算密集型算法)

---

## 🚀 快速启动

1.  **安装环境**：`pip install -r requirements.txt`
2.  **一键运行**：`python run.py`
